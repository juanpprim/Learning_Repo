tracks: knowing when to reach for threads vs. processes vs. a distributed
compute framework vs. a GPU.

2. [`dask-ray-distributed-compute/`](dask-ray-distributed-compute/) — scale pandas-like/general compute across cores and machines.
3. [`gpu-parallelism-cuda-basics/`](gpu-parallelism-cuda-basics/) — SIMD/GPU parallelism fundamentals with numba/CUDA.
4. [`distributed-training/`](distributed-training/) — multi-GPU model training with PyTorch DDP.

Each subtopic is self-contained (README + starter notebook + requirements.txt).
